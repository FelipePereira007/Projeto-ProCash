# ProCash — Aplicativo Bancário UX/UI

O ProCash é um projeto acadêmico desenvolvido para a disciplina de IHC e UX com o objetivo de criar um aplicativo bancário moderno, intuitivo e acessível. O projeto foi baseado nos princípios de Design Thinking, Heurísticas de Nielsen e Design Centrado no Usuário, buscando solucionar problemas reais encontrados em aplicativos financeiros tradicionais.

A aplicação foi desenvolvida utilizando HTML5 e CSS3, contando com telas responsivas, navegação intuitiva, área Pix, dashboard financeiro, perfil do usuário, feedback visual e interface inspirada em bancos digitais modernos.

## Tecnologias utilizadas

* HTML
* CSS
* UX/UI Design
* Design Thinking
* Figma

## Funcionalidades

* Dashboard bancário
* Área Pix
* Histórico financeiro
* Perfil do usuário
* Layout responsivo
* Interface moderna e minimalista

## Objetivo

Desenvolver uma experiência bancária simples, organizada e acessível, focada na usabilidade e na melhor experiência do usuário.
